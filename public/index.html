Welcome to PHPUnit!

PHPUnit is a programmer-oriented testing framework for PHP.
It is an instance of the xUnit architecture for unit testing frameworks.

Take the first steps Read the documentation Learn from presentations Register for training Say "Thank you!"

Current Stable Release

PHPUnit 6.2 is the current stable release series.
It became stable on June 2, 2017.
Support for PHPUnit 6 ends on February 8, 2019.

You can find out what is new in PHPUnit 6.2 in the release announcement.

PHPUnit 6.2 is supported on PHP 7.0 and PHP 7.1.

Download PHPUnit 6.2
(current stable release)

Old Stable Release

PHPUnit 5.7 is the old stable release series.
It became stable on December 2, 2016.
Support for PHPUnit 5 ends on February 2, 2018.

You can find out what was new in PHPUnit 5.7 in the release announcement.

PHPUnit 5.7 is supported on PHP 5.6, PHP 7.0, and PHP 7.1.

Download PHPUnit 5.7
(old stable release)

No Longer Supported

PHPUnit 4.8 is no longer supported.
It became stable on August 7, 2015.
Support for PHPUnit 4 ended on February 3, 2017.

You can find out what was new in PHPUnit 4.8 in the release announcement.

PHPUnit 4.8 was supported on PHP 5.3 – 5.6.

Download PHPUnit 4.8
(no longer supported)

The documentation has details on how to verify PHAR releases of PHPUnit using PGP signatures and SHA1 hashes as well as how to install PHPUnit using Composer.

Old PHAR releases are archived at phar.phpunit.de.

We have a roadmap with details on what is planned for future releases of PHPUnit. Our release process is documented as well.
