Support for major version X ends two years after its initial release (when major version (X+2) is released).

Support for minor version X.Y ends when minor version X.(Y+1) is released.

"Support" means that bugs will be fixed for a version of PHPUnit.

"Life Support" means that changes will be made for an otherwise unsupported version of PHPUnit to be compatible with new versions of PHP (within reason). PHPUnit 9 will be the first version for which "Life Support" is offered after its regular support ends.
